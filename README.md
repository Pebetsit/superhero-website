# 🦸 Superhero Website

## Overview

A responsive multi-page superhero-themed website developed using HTML and CSS. The project presents information about various superheroes through individual profile pages while demonstrating responsive web design and website navigation.

---

## Features

- 🦸 Home page
- 👤 Individual superhero profile pages
- ℹ️ About page
- 📞 Contact page
- 🖼️ Image gallery
- 📱 Responsive layout

---

## Technologies Used

- HTML5
- CSS3

---

## Skills Demonstrated

This project demonstrates:

- Multi-page website development
- HTML page linking
- CSS styling
- Responsive design
- Image handling
- Website navigation
- Folder organization
